libtccw32
=========

the libtcc for win32 only

How to build
============

1. Copy libtccw32/ to your project directory
2. Add libtccw32/src/libtcc.c to your project building source
3. Include libtccw32/src/tcc.h to your `.c' file
4. Build your project
5. Done!
